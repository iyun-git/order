# 餐廳點餐管理系統

這是一個使用 Python 製作的命令列餐廳點餐管理系統，適合作為課堂專題或 Python 基礎練習專案。程式提供餐點選單、建立訂單、結帳付款、庫存扣除、銷售統計，以及刪除最後一筆訂單等功能。

## 專題內容

本專題模擬餐廳櫃台的點餐流程，店員可以透過終端機選單完成顧客點餐與結帳。系統會根據餐點價格自動計算總金額，結帳成功後會扣除庫存並儲存訂單紀錄。後台也可以查看營收、成本、利潤與各餐點銷售數量。

主要功能包含：

- 顯示餐點選單與目前庫存
- 建立顧客訂單
- 檢查輸入格式與庫存數量
- 自動計算應付金額
- 結帳並計算找零
- 訂單完成後自動扣除庫存
- 儲存歷史訂單
- 顯示銷售統計報表
- 刪除最後一筆訂單並恢復庫存

## 執行環境

請先確認電腦已安裝 Python。

建議版本：

```bash
Python 3.8 或以上
```

可以使用以下指令確認 Python 是否已安裝：

```bash
python --version
```

## 如何執行程式

1. 開啟終端機或 PowerShell。

2. 切換到專案資料夾：

```bash
cd "c:\Users\HuiHuinoComputer\Desktop\大三\python\order"
```

3. 執行主程式：

```bash
python order_py.py
```

4. 依照畫面選單輸入功能代號：

```text
1. 顧客新點餐開單
2. 檢視後台營運報表
3. 撤銷上一筆錯誤訂單
0. 關閉並安全退出系統
```

## 操作流程範例

```text
請輸入功能對應數字 (0-3): 1
請輸入餐點編號: 1
請輸入點購數量: 2
請輸入餐點編號: 0
請輸入店員實際收取金額: 200
```
